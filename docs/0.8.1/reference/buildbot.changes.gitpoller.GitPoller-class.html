__init__(self, repourl, branch='master', workdir=None, pollinterval=600, gitbin='git', usetimestamps=True)
(Constructor)

 
Parameters:
  • repourl (string) - the url that describes the remote repository, e.g. git@example.com:foobaz/myrepo.git
  • branch (string) - the desired branch to fetch, will default to 'master'
  • workdir (string) - the directory where the poller should keep its local repository. will default to <tempdir>/gitpoller_work
  • pollinterval (int) - interval in seconds between polls, default is 10 minutes
  • gitbin (string) - path to the git binary, defaults to just 'git'
  • usetimestamps (boolean) - parse each revision's commit timestamp (default True), or ignore it in favor of the current time (to appear together in the waterfall page)
